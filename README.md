# myCLG
Android-Firebase  Application for Universities. 


## Salient Features:-
i) News Feed & Profile Settings
ii) Download Time-Table & Syllabus.
iii) Search Question Papers.
iv) Faculty Contact list
v) eBook Store
vi) Notification on Special announcements
